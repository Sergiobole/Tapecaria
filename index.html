Veludo

Cor/Modelo: #7B3F00

Produtos aconselhados:

  • Limpa Estofados 3M
  • Escova de Cerdas Macias
  • Aspirador Portátil

Manejo e uso: Use o aspirador para remover poeira semanalmente. Aplique o limpa estofados com a escova de cerdas macias, sempre em movimentos suaves e circulares.

Dicas: Evite exposição direta ao sol. Não utilize produtos abrasivos. Para manchas, limpe imediatamente com pano úmido e sabão neutro.

Exemplo: Banco automotivo

Couro

Cor/Modelo: #3E2723

Produtos aconselhados:

  • Hidratante para Couro
  • Pano de Microfibra
  • Limpa Couro

Manejo e uso: Limpe o couro com pano de microfibra levemente umedecido. Aplique o hidratante a cada 2 meses para evitar ressecamento.

Dicas: Não utilize álcool ou solventes. Evite objetos pontiagudos. Mantenha longe do sol para evitar rachaduras.

Exemplo: Poltrona

Suede

Cor/Modelo: #A0522D

Produtos aconselhados:

  • Limpa Suede
  • Escova para Suede
  • Protetor de Tecidos

Manejo e uso: Aspire regularmente. Use a escova para manter a textura. Aplique protetor de tecidos para maior durabilidade.

Dicas: Evite líquidos. Para manchas, utilize limpa suede e seque à sombra.

Exemplo: Cadeira de jantar

Linho

Cor/Modelo: #E5D8B0

Produtos aconselhados:

  • Limpa Tecidos Neutro
  • Escova de Limpeza
  • Aspirador Portátil

Manejo e uso: Aspire semanalmente. Use limpa tecidos neutro para remover manchas leves. Escove suavemente para manter a aparência.

Dicas: Não utilize água em excesso. Evite exposição ao sol. Para manchas difíceis, procure um profissional.
